npm常用命令

发表于 2017-01-12 | 更新于 2020-04-26 | 评论数：

本文总结了一些 npm 的常用命令

项目初始化

npm init

项目生产所需模块 dependencies

npm install <package_name> –save

项目开发和构建所需模块 devDependencies

npm install <package_name> –save-dev

检查安装的模块

npm ls

检查全局安装的模块和依赖

npm ls -g –depth=0

卸载本地模块

npm uninstall <package_name>
如果要在package.json中也删除，则 npm uninstall <package_name> –save，如果你安装的是开发和项目构建包，则npm uninstall <package_name> –save-dev

检查本地模块是否可更新

npm outdated

更新本地模块

npm update

安装全局模块

npm install -g <package_name>

更新全局模块

npm update -g <package_name>

更新全部的全局模块

npm update -g

卸载全局模块

npm uninstall -g <package_name>

帮助

npm help

查看包安装路径

npm root [-g]

npm的配置

npm config

清理模块缓存

npm cache clean

切换npm源

npm install nrm -g 全局安装nrm
nrm ls 显示可用源列表
nrm test 源测速
nrm use 使用某个源
